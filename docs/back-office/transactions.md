# Transactions

## Transactions Table (with Filters):
- Columns: Txn ID, Agent Code, Branch Code, MTO, Type, Amount FCY, Amount PKR, Current State, Status, Created, Last Update, Action.
- Click the action button to view further details.

### Transaction Details:
- Overview: Transaction ID, Bank Reference, Created At, Last Updated, Current State.
- Parties: Remitter, Beneficiary.
- Accounts: Beneficiary account details.
- Amounts & FX: Foreign currency and PKR amount, purpose.
- Screening Summary: Screening status + Completed date.
- Timeline: Stage and lifecycle of the transaction.
