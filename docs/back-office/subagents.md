# Subagents

## 8. Subagents:

### Sub-Agent Management:
<br />
![SubAgent](/img/BO/SubAgents/table.png)
- Add New Sub Agent with details:
  - Sub Agent Name, Description, Head Office Address, Contact Person, Contact Phone, Contact Email, BIPL Settlement Account, Fetch Title, Account Title.

### User Management:
- Add New Sub Agent Users with details: Name, Father Name, ID Number (optional), Email, Phone, Username, Branch Code, Settlement Account, Sub Agent, User Type.
![SubAgent](/img/BO/SubAgents/user.png)

### Bulk Upload:
- Download sample, upload file, and create users in bulk.
- View the bulk upload history for up to a month.
![SubAgent](/img/BO/SubAgents/Bulk.png)

### Sub-Agent Maker Inbox:
- View subagent creation and bulk request creation, edit requests, update, and remove requests.

### Sub-Agent Checker Inbox:
- Approve or reject subagent creation with reason required.

### User Maker Inbox:
- View subagent user creation and bulk request creation, edit requests, update, and remove requests.

### User Checker Inbox:
- Approve or reject subagent user creation with reason required.
