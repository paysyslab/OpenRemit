# Branch Records

## 6. Branch Records:

### User Management:

<br /> 
![User Management](/img/BO/BranchUsers/UM.png)
- Add new user records with the following details:
  - Full Name, Username (lowercase, letters, numbers, and dots only), Email Address, Mobile Number (starts with +92), Branch Code (optional), Settlement Account (optional), Designation (optional).
- View user details with columns: Full Name, Username, Email, Mobile, Branch Code, Settlement Account, Designation, Currency, Status, Created At, Actions (View, Edit, Sync from Bank, Activate/Deactivate, Reset Password, Password Reset History, Audit Log).
