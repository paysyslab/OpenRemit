# Logging In and Changing Password

## 1. Branch User:

### Login:
- User logs in with:
  - User ID
  - Password
  - Branch Code (4 digits)
- Receives OTP on email
- Checks 'Trust This Device'
- Redirected to change password on first-time login on every new device
- After changing password, logs in again with new credentials and lands on dashboard

### Changing Password:
- User chooses to change password and is redirected to a screen where they enter their old and new password.
- They're redirected to login screen and they log in with new creds.

### Dashboard:
- Can view stats with filters
