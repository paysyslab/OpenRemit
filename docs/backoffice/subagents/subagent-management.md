## Objective
This page covers creating and managing sub-agent records.

![SubAgent](/img/subAgents/subAgentManagement.png)

## Add New Sub-Agent (Maker)
Navigate to **Backoffice → Sub-Agents → Sub-Agent Management → Add New Sub Agent** and fill:

- **Sub Agent Name** (mandatory)
- **Description** (mandatory)
- **Head Office Address** (mandatory)
- **Contact Person** (mandatory)
- **Contact Phone** (mandatory)
- **Contact Email** (mandatory)
- **BIPL Settlement Account** (mandatory)
- **Fetch Title** (setting)
- **Account Title** (mandatory when applicable)

![SubAgent](/img/subAgents/subAgentMaker.png)

## Approval
Sub-agent records require Checker approval.


![SubAgent](/img/subAgents/SubAgentChecker.png)
