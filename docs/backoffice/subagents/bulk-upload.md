## Objective
Bulk User Upload allows onboarding multiple sub-agent users via file upload.

## Steps
1. Navigate to **Backoffice → Sub-Agents → Bulk User Upload**.
2. Download the sample/template file (if provided).
3. Populate user records.
4. Upload file and review validation results.
5. Submit for approval (if Maker/Checker is enabled).

## Common Validation Errors
- Invalid username format;
- Missing branch code;
- Incorrect settlement account format;
- Missing required user fields.
