## Objective
Create and manage sub-agent users who operate at branch/user level.

## Add New Sub-Agent User (Maker)
Navigate to **Backoffice → Sub-Agents → User Management → Add New Sub Agent Users**:

Fields:
- **Name** (mandatory)
- **Father Name** (mandatory)
- **ID Number** (optional)
- **Email** (mandatory)
- **Phone** (mandatory)
- **Username** (mandatory)
- **Branch Code** (mandatory)
- **Settlement Account (Branch/User)** (mandatory)
- **Sub Agent** (mandatory)
- **User Type** (mandatory)
