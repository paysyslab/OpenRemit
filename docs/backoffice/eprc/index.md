## Objective
The e-PRC module supports creation and download of the Electronic Proceeds Realization Certificate (e-PRC) for eligible remittance transactions.

## What is e-PRC
An e-PRC certifies that:
- remittance was received by order of the remitter; and
- the equivalent amount in PKR was paid to the beneficiary.

It is a system-generated document; no signature is required.

## Eligibility 
Eligibility rules can vary by deployment, but commonly:
- transaction must be successfully realized/posted; or
- transaction must meet reporting criteria.

## Key Capabilities
- View and download e-PRC for a single transaction;
- Bulk download e-PRCs for multiple transactions;
- Verify e-PRC fields for SBP reporting alignment.

---

## Objective
This page explains how to view and download e-PRC documents for single transactions and in bulk.

## Single Transaction e-PRC
1. Locate the transaction via **Transactions** search.
2. Open **Actions** and select **e-PRC** (or navigate to e-PRC module if separate search exists).
3. Confirm details and download the e-PRC.

## Bulk e-PRC Download
1. Filter transactions by date range, partner, and status.
2. Select multiple transactions (checkbox selection).
3. Choose **Bulk Download e-PRC**.
4. Download the generated archive or consolidated output as provided by your deployment.

## e-PRC Fields (Reference)
An e-PRC typically includes:

- e-PRC No.; Date of e-PRC;
- Remitter name and unique identifier (if available);
- Beneficiary name, bank, and IBAN (if applicable);
- Remitting institution name (e.g., RIA Money Transfer);
- Country name and code (as per ITRS);
- Realization date;
- FCY amount; conversion rate; PKR amount;
- Purpose code (ITRS) and purpose description;
- Transaction reference number;
- Reporting statement for SBP/SBP-BSC ITRS schedule.
