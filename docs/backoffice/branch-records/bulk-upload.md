## Objective
Bulk Uploads allows mass creation of branch users through an upload file.

![authentication flow](/img/bulkUpload.png)

## Steps
1. Navigate to **Backoffice → Branch Records → Bulk Uploads**.
2. Download the **Sample File**.
3. Populate user records in the required format.
4. Upload the completed file.
5. Review validation results.

## Processing Behavior
Depending on configuration:
- valid rows are accepted and submitted for approval;
- invalid rows are rejected with error details.

## Best Practices
- Keep file format identical to sample.
- Validate emails and phone numbers before upload.
- Avoid duplicate usernames.
