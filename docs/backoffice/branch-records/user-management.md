## Objective
User Management allows authorized users to view existing branch users and create new branch users.

## View Approved Users
- Navigate to **Backoffice → Branch Records → User Management**.
- Review the approved users list.
- Use search and filters if available.

## Create a New User (Maker)
1. Click **Create User**.
2. Enter the following fields:
   - **Full Name** (mandatory)
   - **Username** (mandatory; lowercase letters, numbers, and dots only)
   - **Email Address** (mandatory)
   - **Mobile Number** (mandatory)
   - **Branch Code** (optional)
   - **Settlement Account** (optional)
   - **Designation** (optional)

![authentication flow](/img/branchRecords/userCreation.png)   

3. Submit the request.

## Approval Flow
- If Maker/Checker is enabled, the user is not active until approved by the Checker.
- Rejected requests must be corrected and resubmitted.

## Validation Guidance
- Ensure usernames follow naming rules strictly.
- Use corporate email where required by policy.
- Ensure branch code and settlement account are accurate if provided.
