## Objective
Branch Records provides branch-level user administration and operational governance with Maker/Checker controls.

## What You Can Do
- View approved users;
- Create users (Maker);
- Bulk upload user creation requests;
- Review and approve/reject requests (Checker).

## Typical Roles
- **Maker**: creates or uploads users and submits for approval;
- **Checker**: validates information and approves or rejects.
