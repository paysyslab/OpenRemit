## Objective
This inbox supports dual control for partner and partner user onboarding.

## Partner Maker Inbox
Used to:
- view rejected partner requests;
- correct partner details;
- resubmit for approval.

![authentication flow](/img/partner/partnerMakerInbox.png)

## Partner User Maker Inbox
Used to:
- view rejected partner-user requests;
- correct user details;
- resubmit for approval.

![authentication flow](/img/partner/userMakerInbox.png)

## Checker Actions
Checkers should verify:
- partner settlement mapping;
- screening and title match configuration correctness;
- completeness of contact and address details;
- correctness of user-to-partner mapping.

Decisions are logged for audit.
