## Objective
Partner User Management allows creation and maintenance of users that operate on behalf of a partner.

## Add New Partner User (Maker)
Navigate to **Backoffice → Partners → User Management → Add New Partner User**:

![authentication flow](/img/partner/AddNewPartnerUser.png)

Fields:
- **Name** (mandatory)
- **Father Name** (mandatory)
- **ID Number** (optional)
- **Email** (mandatory)
- **Phone** (mandatory)
- **Username** (mandatory)
- **Partner** (mandatory)
- **User Type** (mandatory)

## Approval
The user request is routed to Checker for approval.

![authentication flow](/img/partner/userChecker.png)
