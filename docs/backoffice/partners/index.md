## Objective
Partners represent integrated entities (e.g., MTOs) sending remittance instructions. The Partners module controls partner master data, integration modes, and validation toggles.

## Core Capabilities
- Maintain partner profile and settlement information;
- Configure per-mode checks (OTC/FT/IBFT);
- Manage partner users;
- Process Maker/Checker approvals for onboarding changes.
